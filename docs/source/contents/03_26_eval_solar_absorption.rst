========================================================================================================================
日射吸収量
========================================================================================================================

------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、家具と境界における日射吸収量の計算方法について定義する。

本節の計算は、次の値を必要とする。

    1. ステップ |n| における室 |i| の透過日射量, W
    2. 室と境界との関係を表す係数 :math:`p_{ij}`
    3. 境界 |j| の面積 :math:`A_{s,j}`, |m2|
    4. 境界 |j| が透過日射を吸収するか否かを表す論理値 :math:`B_{s,sol,abs,j}`

本節の計算の結果、次の値が得られる。

    1. ステップ |n| からステップ |n+1| における室 |i| に設置された家具による透過日射吸収熱量の時間平均値 :math:`q_{sol,frt,i,n}`
    2. ステップ |n| における境界 |j| の透過日射吸収熱量 :math:`q_{s,sol,j,n}`

------------------------------------------------------------------------------------------------------------------------
2 記号及び添え字
------------------------------------------------------------------------------------------------------------------------

この計算で用いる記号及び単位を次に示す。

.. list-table:: 表1 記号及び単位
    :header-rows: 1
    :widths: 1,6,1

    * - 記号
      - 意味
      - 単位
    * - :math:`A_{s}`
      - 境界の面積
      - W / |m2|
    * - :math:`A_{s,abs}`
      - ある室の日射が吸収される境界の面積の合計
      - |m2|
    * - :math:`B_{s,sol,abs}`
      - ある境界が透過日射を吸収するか否かを表す論理値
      - －
    * - :math:`q_{s,sol}`
      - ある境界の透過日射吸収熱量
      - W / |m2|
    * - :math:`q_{sol,frt}`
      - ある室に設置された家具による透過日射吸収熱量の時間平均値
      - W
    * - :math:`q_{trs,sol}`
      - 室の透過日射量
      - W
    * - :math:`p_{ij}`
      - 境室と境界との関係を表す係数
      - －
    * - :math:`r_{sol,frt}`
      - 室内侵入日射のうち家具に吸収される割合
      - ー

------------------------------------------------------------------------------------------------------------------------
3 室に設置された家具による透過日射吸収熱量の時間平均値
------------------------------------------------------------------------------------------------------------------------

ステップ |n| からステップ |n+1| における室 |i| に設置された家具による透過日射吸収熱量の時間平均値 :math:`q_{sol,frt,i,n}` は次式により表される。

.. math::
    :nowrap:

    \begin{align*}
        q_{sol,frt,i,n} = q_{trs,sol,i,n} \cdot r_{sol,frt}
        \tag{1}
    \end{align*}

|   :math:`q_{sol,frt,i,n}` : ステップ |n| からステップ |n+1| における室 |i| に設置された家具による透過日射吸収熱量の時間平均値, W
|   :math:`q_{trs,sol,i,n}` : ステップ |n| における室 |i| の透過日射量, W
|   :math:`r_{sol,frt}` : 室内侵入日射のうち家具に吸収される割合, -

------------------------------------------------------------------------------------------------------------------------
4 境界の透過日射吸収熱量
------------------------------------------------------------------------------------------------------------------------

ステップ |n| における境界 |j| の透過日射吸収熱量 :math:`q_{s,sol,j,n}` は次式により表される。

:math:`B_{s,sol,abs,j} = True` のとき

.. math::
    :nowrap:

    \begin{align*}
        q_{s,sol,j,n} = q_{trs,sol,i,n} / A_{s,abs,i} \cdot p_{ij,i,j} \cdot ( 1 - r_{sol,frt} )
        \tag{2a}
    \end{align*}

:math:`B_{s,sol,abs,j} = False` のとき

.. math::
    :nowrap:

    \begin{align*}
        q_{s,sol,j,n} = 0
        \tag{2b}
    \end{align*}

|   :math:`q_{s,sol,j,n}` : ステップ |n| における境界 |j| の透過日射吸収熱量, W / |m2|
|   :math:`q_{trs,sol,i,n}` : ステップ |n| における室 |i| の透過日射量, W
|   :math:`A_{s,abs,i}` : 境界 |j| の面積, |m2|
|   :math:`p_{ij,i,j}` : 室 |i| と境界 |j| との関係を表す係数, -
|   :math:`r_{sol,frt}` : 室内侵入日射のうち家具に吸収される割合, -
|   :math:`A_{s,abs,i}` : 室 |i| の日射が吸収される境界の面積の合計, |m2|
|   :math:`B_{s,sol,abs,j}` : 境界 |j| が透過日射を吸収するか否かを表す論理値

室 |i| の日射が吸収される境界の面積の合計 :math:`A_{s,abs,i}` は次式により表される。

.. math::
    :nowrap:

    \begin{align*}
        A_{s,abs,i} = \sum_{j \in \{ j \mid B_{s,sol,abs,j} = True \} }{ p_{ij,i,j} \cdot A_{s,j} }
        \tag{3}
    \end{align*}

室内侵入日射のうち家具に吸収される割合 :math:`r_{sol,frt}` は次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        r_{sol,frt} = 0.5
        \tag{4}
    \end{align*}
