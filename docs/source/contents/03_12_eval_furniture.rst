========================================================================================================================
備品等
========================================================================================================================


------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、室に付属する家具等の備品等の仕様について決定する。

本節の計算は、次の値を必要とする。

    1. 備品等の指定方法（直接指定するか、室の気積から計算されたデフォルト値を仕様するか）
    2. 室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}` (備品等の仕様を直接指定する場合)
    3. 室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}` (備品等の仕様を直接指定する場合)
    4. 室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}` (備品等の仕様を直接指定する場合)
    5. 室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}` (備品等の仕様を直接指定する場合)

本節で示す計算の結果、次の値が得られる。

    1. 室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}`
    2. 室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}`
    3. 室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}`
    4. 室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}`

3 備品等の仕様
------------------------------------------------------------------------------------------------------------------------

室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}` は、直接指定するか、式(1)で表される。

.. math::
    :nowrap:

    \begin{align*}
        G_{lh,frt,i} = f_{g,lh,frt} \cdot C_{lh,frt,i} \tag{1}
    \end{align*}

ここで、

:math:`G_{lh,frt,i}` : 室 |i| の空気と備品等間の湿気コンダクタンス, kg / ( s kg / kg(DA) )

:math:`f_{g,lh,frt}` : 備品等の湿気容量あたりの空気との間の湿気コンダクタンスを表す係数, 1 / s

:math:`C_{lh,frt,i}` : 室 |i| の備品等の湿気容量, kg / ( kg / kg(DA) )

である。

備品等の湿気容量あたりの空気との間の湿気コンダクタンスを表す係数 :math:`f_{g,lh,frt}` は、 :math:`0.0018` 1 / s とする。

室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}` は、直接指定するか、式(2)で表される。

.. math::
    :nowrap:

    \begin{align*}
        C_{lh,frt,i} = f_{c,lh,frt} \cdot V_{rm} \tag{2}
    \end{align*}

ここで、

:math:`C_{lh,frt,i}` : 室 |i| の備品等の湿気容量, kg / ( kg / kg(DA) )

:math:`f_{c,lh,frt}` : 室の気積あたりの備品等の湿気容量を表す係数, kg / ( |m3| kg / kg(DA) )

:math:`V_{rm}` : 室 |i| の気積, |m3|

である。

室の気積あたりの備品等の湿気容量を表す係数 :math:`f_{c,lh,frt}` は、一般の家具等を想定し、 :math:`16.8` kg / ( |m3| kg / kg(DA) ) とする。

室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}` は、直接指定するか、式(3)で表される。

.. math::
    :nowrap:

    \begin{align*}
        G_{sh,frt,i} = f_{g,sh,frt} \cdot  C_{sh,frt,i} \tag{3}
    \end{align*}

ここで、

:math:`G_{sh,frt,i}` : 室 |i| の空気と備品等間の熱コンダクタンス, W / K

:math:`f_{g,sh,frt}` : 備品等の熱容量あたりの空気との間の熱コンダクタンスを表す係数, 1 / s

:math:`C_{sh,frt,i}` : 室 |i| の備品等の熱容量, J / K

である。

備品等の熱容量あたりの空気との間の熱コンダクタンスを表す係数 :math:`f_{g,sh,frt}` は、 :math:`0.00022` 1 / s とする。

室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}` は、直接指定するか、式(4)で表される。

.. math::
    :nowrap:

    \begin{align*}
        C_{sh,frt,i} = f_{c,sh,frt} \cdot V_{rm} \tag{4}
    \end{align*}

ここで、

:math:`C_{sh,frt,i}` : 室 |i| の備品等の熱容量, J / K

:math:`f_{c,sh,frt}` : 室の気積あたりの備品等の熱容量を表す係数, J / (|m3| K)

:math:`V_{rm}` : 室 |i| の気積, |m3|
    
である。

室の気積あたりの備品等の熱容量を表す係数 :math:`f_{c,sh,frt}` は、一般の家具等を想定し、 :math:`12.6 \times 10^3` J / ( |m3| K ) とする。
