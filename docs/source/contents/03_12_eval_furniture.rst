========================================================================================================================
備品等
========================================================================================================================


------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、室に付属する家具等の備品等の仕様について決定する。

本節の計算は、次の値を必要とする。

    1. 入力ファイルにおける room 要素 の furniture 要素
    2. 室 |i| の容積

本節で示す計算の結果、次の値が得られる。

    1. 室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}`
    2. 室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}`
    3. 室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}`
    4. 室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}`
    5. 室 |i| の備品等の日射吸収割合 :math:`r_{sol,frt,i}`

3 熱容量・湿気容量・熱コンダクタンス・湿気コンダクタンス
------------------------------------------------------------------------------------------------------------------------

室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}` 、
室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}` 、
室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}` および
室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}` は、
room 要素の furniture 要素の input_method により計算方法が異なる。

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
3.1 input_method が 'specify' の場合
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}` は、room 要素の furniture 要素の heat_capacity の値（単位： J / K ）とする。

室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}` は、room 要素の furniture 要素の moisture_capacity の値（単位： kg / (kg / kgDA) ）とする。

室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}` は、room 要素の furniture 要素の heat_cond の値（単位： W / K ）とする。

室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}` は、room 要素の furniture 要素の moisture_cond の値（単位： kg / (s (kg / kgDA) ) ）とする。

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
3.2 input_method が 'default' の場合
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

室 |i| の空気と備品等間の湿気コンダクタンス :math:`G_{lh,frt,i}` は、次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        G_{lh,frt,i} = f_{g,lh,frt} \cdot C_{lh,frt,i} \tag{1}
    \end{align*}

備品等の湿気容量あたりの空気との間の湿気コンダクタンスを表す係数 :math:`f_{g,lh,frt}` は、 :math:`0.0018` 1 / s とする。

室 |i| の備品等の湿気容量 :math:`C_{lh,frt,i}` は、次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        C_{lh,frt,i} = f_{c,lh,frt} \cdot V_{rm} \tag{2}
    \end{align*}

室の気積あたりの備品等の湿気容量を表す係数 :math:`f_{c,lh,frt}` は、一般の家具等を想定し、 :math:`16.8` kg / ( |m3| kg / kg(DA) ) とする。

室 |i| の空気と備品等間の熱コンダクタンス :math:`G_{sh,frt,i}` は、次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        G_{sh,frt,i} = f_{g,sh,frt} \cdot  C_{sh,frt,i} \tag{3}
    \end{align*}

備品等の熱容量あたりの空気との間の熱コンダクタンスを表す係数 :math:`f_{g,sh,frt}` は、 :math:`0.00022` 1 / s とする。

室 |i| の備品等の熱容量 :math:`C_{sh,frt,i}` は、次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        C_{sh,frt,i} = f_{c,sh,frt} \cdot V_{rm} \tag{4}
    \end{align*}

室の気積あたりの備品等の熱容量を表す係数 :math:`f_{c,sh,frt}` は、一般の家具等を想定し、 :math:`12.6 \times 10^3` J / ( |m3| K ) とする。

------------------------------------------------------------------------------------------------------------------------
4 備品等の日射吸収割合
------------------------------------------------------------------------------------------------------------------------

室 |i| の備品等の日射吸収割合 :math:`r_{sol,frt,i}` は、
room 要素の furniture 要素の solar_absorption_ratio 要素が存在する場合はその値（単位：無次元）とし、
要素が存在しない場合は、規定値の :math:`0.5` とする。
