========================================================================================================================
日よけの影面積比率
========================================================================================================================

------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、日よけの影面積比率の求め方について定義する。

日よけの影面積比率を求める方法は、横方向に無限に長いオーバーハング型日よけの評価方法（以下、簡易法という）と、
開口部上部の横方向が有限のひさし(オーバーハング)と開口部側方のひさし(サイドフィン)の組み合わせが評価できる
評価方法（以下、詳細法という）の2種類ある。

本節の計算は、次の値を必要とする。

    1. ステップ |n| における太陽高度 :math:`h_{sun,n}`
    2. ステップ |n| における太陽方位角 :math:`a_{sun,n}`
    3. 開口部 |j| の方位角 :math:`\alpha_{w,j}`
    4. 開口部 |j| のひさしの出幅 :math:`l_{z,j}` （簡易法のみ）
    5. 開口部 |j| の高さ :math:`l_{y,h,j}` （簡易法のみ）
    6. 開口部 |j| の上端から日よけまでの長さ :math:`l_{y,e,j}` （簡易法のみ）
    7. （詳細法については記述を省略）

本節で示す計算の結果、次の値が得られる。

    1. ステップ |n| における開口部 |j| の直達日射に対する日よけの影面積比率 :math:`f_{ss,d,j,n}`
    2. 開口部 |j| の天空放射に対する日よけの影面積比率 :math:`f_{ss,s,j}`
    3. 開口部 |j| の地盤反射に対する日よけの影面積比率 :math:`f_{ss,r,j}`

なお、簡易法における各長さは次図のとおりである。

4 開口部上部に横方向に無限にのびるひさしがある場合
------------------------------------------------------------------------------------------------------------------------

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
4.1 開口部の直達日射に対する日よけの影面積比率
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

ステップ |n| における開口部 |j| の直達日射に対する日よけの影面積比率 :math:`f_{ss,d,j,n}` は、次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        f_{ss,d,j,n} = l_{ss,d,y,j,n} / l_{y,h,j} \tag{1}
    \end{align*}

ただし、:math:`f_{ss,d,j,n}` が :math:`1.0` を超える場合は :math:`f_{ss,d,j,n} = 1.0` とする。
:math:`f_{ss,d,j,n}` が :math:`0.0` を下回る場合は :math:`f_{ss,d,j,n} = 0.0` とする。


また、 :math:`h_{sun,n} \le 0` の場合（太陽高度が0以下の場合）
および :math:`\cos(a_{s,n} - \alpha_{w,j}) \le 0` の場合（太陽位置が境界から見て見えている場合）は、
:math:`f_{ss,d,j,n} = 0.0` とする。

なお、直達日射に対する日よけの影面積比率 :math:`f_{ss,d,j,n}` が :math:`0` となるのは窓上端に日陰がかからない場合であり、 :math:`1` となるのは窓下まで影になる場合である。

ステップ |n| に開口部 |j| に影がかかる長さ(窓上端から下方への長さ) :math:`l_{ss,d,y,j,n}` は、次式で表される。

.. math::
    :nowrap:

    \begin{align*}
        l_{ss,d,y,j,n} = l_{z,j} \cdot \tan \phi_{j,n} - l_{y,e,j} \tag{2}
    \end{align*}

ステップ |n| における開口部 |j| に対する太陽のプロファイル角 :math:`\phi_{j,n}` の正接は、式(3)で表される。

.. math::
    :nowrap:

    \begin{align*}
        \tan \phi_{j,n} = \tan h_{sun,n} / \cos ( a_{sun,n} - \alpha_{w,j} )
    	\tag{3}
    \end{align*}

ただし、ステップ |n| において日が沈んでいる場合は、 :math:`h_{sun,n}=0` 及び :math:`a_{sun,n}=0` とする。

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
4.2 開口部の天空放射に対する日よけの影面積比率
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

開口部 |j| の天空放射に対する日よけの影面積比率 :math:`f_{ss,s,j}` は、式(2)で表される。

.. math::
    :nowrap:

    \begin{align*}
        f_{ss,s,j} = \dfrac{
            \left\{ (l_{y,e,j} + l_{y,h,j}) + \sqrt{ {l_{y,e,j}}^2 + {l_{z,j}}^2 } \right\} - \left\{ l_{y,e,j} + \sqrt{ (l_{y,e,j} + l_{y,h,j})^2 + {l_{z,j}}^2 } \right\} } {2 \cdot l_{y,h,j}
        }
        \tag{4}
    \end{align*}

なお、天空放射に対する日よけによる低減率 :math:`f_{ss,s,j}` は、ひさしが無い場合で :math:`0` を、ひさしが無限に出ている仮想の場合で :math:`0.5` の値をとる。

^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
