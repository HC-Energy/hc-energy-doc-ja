========================================================================================================================
プログラムの使い方
========================================================================================================================

プログラムの使い方を示します。

.. code-block::
    :caption: プログラムの使い方

    python3 src/heat_load_calc/heat_load_calc.py arg1 [-o arg2] [--schedule_saved] [--weather_saved] [--log arg3]


arg1
    | 必須の引数です。
    | 入力ファイルを指定します。
    | ルートからの相対パスで指定します。
    | 例： src/heat_load_calc/input_files/example.json
[-o arg2]
    | 代替コマンド --output_data_dir
    | arg2 には計算結果を出力するディレクトリを指定します。
    | 例： src/heat_load_calc/output_files/
    | 計算結果は2つのCSVファイルで出力されます。
    | この引数を省略した場合は、ルートフォルダ（src/）に出力されます。
[--schedule_saved]
    | この引数を指定した場合、計算に用いられたスケジュールファイルが出力されます。
    | この引数を省略した場合、スケジュールファイルは出力されません。
[--weather_saved]
    | この引数を指定した場合、計算に用いられた気象データファイルが出力されます。
    | この引数を省略した場合、気象データファイルは出力されません。
[--log arg3]
    | 計算時にログを出すレベルを指定します。
    | arg3 には次の値を指定します。
    | DEBUG / INFO / WARN / ERROR / CRITICAL
    | この引数を省略した場合、ERRORが指定されたとみなします。
