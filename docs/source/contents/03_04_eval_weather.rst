========================================================================================================================
気象条件
========================================================================================================================

------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、気象条件の指定の仕方について定義する。

気象条件の仕方は、任意の地点におけるデータを独自に用意するか又は建築物省エネ法における地域の区分を指定する。

任意の地点におけるデータを独自に用意する場合、本節の計算は、次の値を必要とする。

    1. 時間間隔（15分間隔・30分間隔・1時間間隔）
    2. 緯度, 度
    3. 経度, 度
    4. 年間の1時間ごとの外気温度, ℃
    5. 年間の1時間ごとの外気絶対湿度, kg / kg(DA)
    6. 年間の1時間ごとの法線面直達日射量, W / |m2|
    7. 年間の1時間ごとの水平面全天日射量, W / |m2|
    8. 年間の1時間ごとの夜間放射量, W / |m2|

なお、ここで準備される年間の1時間ごとのデータの個数は8760である。

建築物省エネ法における地域の区分を指定する場合、本節の計算は、次の値を必要とする。

    1. 時間間隔（15分間隔・30分間隔・1時間間隔）
    2. 地域の区分

本節で示す計算の結果、次の値が得られる。

    1. ステップ |n| における外気温度, :math:`\theta_{o,n}`, ℃
    2. ステップ |n| における外気絶対湿度, :math:`X_{o,n}`, kg / kg(DA)
    3. ステップ |n| における法線面直達日射量, :math:`I_{dn,n}`, W / |m2|
    4. ステップ |n| における水平面全天日射量, :math:`I_{sky,n}`, W / |m2|
    5. ステップ |n| における夜間放射量, :math:`R_{n,n}`, W / |m2|
    6. ステップ |n| における太陽方位角 :math:`a_{sun,n}`, rad
    7. ステップ |n| における太陽高度 :math:`h_{sun,n}`, rad

------------------------------------------------------------------------------------------------------------------------
3 気象条件
------------------------------------------------------------------------------------------------------------------------

年間の1時間ごとの外気温度 :math:`\theta_{o,h}`, 
年間の1時間ごとの外気絶対湿度 :math:`X_{o,h}`, 
年間の1時間ごとの法線面直達日射量 :math:`I_{dn,h}`, 
年間の1時間ごとの水平面全天日射量 :math:`I_{sky,h}` 及び 
年間の1時間ごとの夜間放射量 :math:`R_{n,h}` は、
これらの値を直接指定するか又は地域の区分に応じて用意されたデータを参照する。

年間の1時間ごとの太陽方位角 :math:`a_{sun,h}` 及び
年間の1時間ごとの太陽高度 :math:`h_{sun,h}` は、緯度・経度に応じて「太陽位置」に規定する方法によって計算する。
緯度・経度は直接指定するか、建築物省エネ法における地域の区分が指定された場合は「地域」に規定する方法によって計算する。

これらの値は1月1日0時の値から12月31日23時までの8760個のデータで与えられる。

一方で、12月31日23時以降の計算において12月31日24時のデータを必要とすることから、12月31日24時の値は1月1日0時の値に等しいとする。

.. math::
    :nowrap:

    \begin{align*}
        D_h \vert_{h=8760} = D_h \vert_{h=0}
    \end{align*}

ステップ :math:`n` における値については、次式のように時間間隔に応じて内挿する。

時間間隔が1時間の場合、

.. math::
    :nowrap:

    \begin{align*}
        D_n \vert_{n = h} = D_h
    \end{align*}

時間間隔が30分の場合、

.. math::
    :nowrap:

    \begin{align*}
        D_n \vert_{n = 2 h} = D_h
    \end{align*}

    \begin{align*}
        D_n \vert_{n = 2 h + 1} = \frac{ D_h + D_{h+1} }{2}
    \end{align*}

時間間隔が15分の場合、

.. math::
    :nowrap:

    \begin{align*}
        D_n \vert_{n = 4 h} = D_h
    \end{align*}

    \begin{align*}
        D_n \vert_{n = 4 h + 1} = \frac{ 3 D_h + D_{h+1} }{4}
    \end{align*}

    \begin{align*}
        D_n \vert_{n = 4 h + 1} = \frac{ 2 D_h + 2 D_{h+1} }{4}
    \end{align*}

    \begin{align*}
        D_n \vert_{n = 4 h + 1} = \frac{ D_h + 3 D_{h+1} }{4}
    \end{align*}
