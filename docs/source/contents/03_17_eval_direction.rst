========================================================================================================================
傾斜面の方位角・傾斜角
========================================================================================================================

------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、境界の向きに応じて境界の方位角・傾斜角の求め方について定義する。

本節の計算は、次の値を必要とする。

    1. 境界の向き

本節で示す計算の結果、次の値が得られる。

    1. 境界 |j| の方位角 :math:`\alpha_{w,j}`, rad
    2. 境界 |j| の傾斜角 :math:`\beta_{w,j}`, rad

------------------------------------------------------------------------------------------------------------------------
3 方位角
------------------------------------------------------------------------------------------------------------------------

境界 |j| の方位角 :math:`\alpha_{w,j}` は、境界 |j| の向きに応じて、以下のように定義される。

* 南を向いている壁の場合： :math:`\alpha_{w,j} = 0`
* 南西を向いている壁の場合： :math:`\alpha_{w,j} = \pi/4`
* 西を向いている壁の場合： :math:`\alpha_{w,j} = \pi/2`
* 北西を向いている壁の場合： :math:`\alpha_{w,j} = 3 \pi/4`
* 北を向いている壁の場合： :math:`\alpha_{w,j} = \pi`
* 北東を向いている壁の場合： :math:`\alpha_{w,j} = - 3 \pi/4`
* 東を向いている壁の場合： :math:`\alpha_{w,j} = - \pi/2`
* 南東を向いている壁の場合： :math:`\alpha_{w,j} = - \pi/4`
* 上を向いている屋根・天井の場合： 定義しない
* 下を向いている床の場合： 定義しない

------------------------------------------------------------------------------------------------------------------------
4 傾斜角
------------------------------------------------------------------------------------------------------------------------

境界 |j| の傾斜角 :math:`\beta_{w,j}` は、境界 |j| の向きに応じて、以下のように定義される。

* 上を向いている屋根・天井の場合： :math:`\beta_{w,j} = 0`
* 壁の場合： :math:`\beta_{w,j} = \pi/2`
* 下を向いている床の場合： :math:`\beta_{w,j} = \pi`
