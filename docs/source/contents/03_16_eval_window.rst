------------------------------------------------------------------------------------------------------------------------
1 はじめに
------------------------------------------------------------------------------------------------------------------------

本節では、窓のパラメータの計算方法について記述する。

本節の計算の結果、次の項目が得られる。

    1. ステップ |n| における境界 |j| の窓の直達日射に対する日射熱取得率 :math:`\eta_{w,d,j,n}`, -
    2. ステップ |n| における境界 |j| の窓の直達日射に対する日射透過率 :math:`\eta_{w,d,j,n}`, -
    3. 境界 |j| の窓の天空放射に対する日射熱取得率 :math:`\eta_{w,s,j}`, -
    4. 境界 |j| の窓の地面反射光に対する日射熱取得率 :math:`\eta_{w,r,j}`, -
    5. 境界 |j| の窓の天空放射に対する日射透過率 :math:`\tau_{w,s,j}`, -
    6. 境界 |j| の窓の地面反射光に対する日射透過率 :math:`\tau_{w,r,j}`, -

本節の計算は次の値を必要とする。

    1. 境界 |j| の窓の熱損失係数(U値) :math:`U_{w,j}`, W / |m2| K
    2. 境界 |j| の日射熱取得率(η値) :math:`\eta_{w,j}`, -
    3. 境界 |j| の窓の面積に対するグレージングの面積の比 :math:`r_{A,w,g,j}`, -
    4. 境界 |j| の窓の建具(フレーム)材質の種類(樹脂・アルミニウム・アルミ樹脂複合) :math:`T_{w,flame}`
    5. 境界 |j| の窓の構成(単層・複層) :math:`T_{w,glass}`
    6. ステップ |n| における境界 |j| の窓に対する直達日射の入射角 :math:`\varphi_{j,n}`, rad

５．計算仕様書

(3) 計算方法
ステップnにおける境界jの窓の直達日射に対する日射熱取得率η_(w,d,j,n)及びステップnにおける境界jの窓の直達日射に対する日射透過率η_(w,d,j,n)は次式により表される。
η_(w,j,n)=η_(w,j) (φ_n )
τ_(w,j,n)=τ_(w,j) (φ_n )
η_(w,j,n)	：ステップnにおける境界jの窓の日射熱取得率
τ_(w,j,n)	：ステップnにおける境界jの窓の日射透過率
η_(w,j) (φ)	：入射角φに対する境界jの窓の日射熱取得率（関数）
τ_(w,j) (φ)	：入射角φに対する境界jの窓の日射透過率（関数）
φ_n	：ステップnにおける直達日射の入射角, rad
境界jの窓の天空放射に対する日射熱取得率η_(w,s,j)・境界jの窓の地面反射光に対する日射熱取得率η_(w,r,j)・境界jの窓の天空放射に対する日射透過率η_(w,s,j)・境界jの窓の地面反射光に対する日射透過率τ_(w,r,j)は次式により表される。
η_(w,s,j)=η_(w,c,j)
η_(w,r,j)=η_(w,c,j)
τ_(w,s,j)=τ_(w,c,j)
τ_(w,r,j)=τ_(w,c,j)
η_(w,c,j)=π/M⋅∑_(m=1)^M▒〖η_(w,j) (φ_m )⋅sinφ_m⋅cosφ_m 〗
τ_(w,c,j)=π/M⋅∑_(m=1)^M▒〖τ_(w,j) (φ_m )⋅sinφ_m⋅cosφ_m 〗
ここで、
φ_m=π/2⋅(m-1/2)/M
m=1,…,M
M=1000
とする。
入射角φに対する境界jの窓の日射熱取得率η_(w,j) (φ)及び入射角φに対する境界jの窓の日射透過率τ_(w,j) (φ)は次式により表される。
η_(w,j) (φ)=η_(w,g,j) (φ)⋅r_(A,w,g,j)
τ_(w,j) (φ)=τ_(w,g,j) (φ)⋅r_(A,w,g,j)
η_(w,j)	：入射角φに対する境界jの窓の日射熱取得率, -
η_(w,g,j)	：入射角φに対する境界jの窓のガラス部分の日射熱取得率, -
τ_(w,j)	：入射角φに対する境界jの窓の日射熱取得率, -
τ_(w,g,j)	：入射角φに対する境界jの窓のガラス部分の日射熱取得率, -
r_(A,w,g,j)	：境界jの窓の面積に対するグレージングの面積の比, -

入射角φに対する境界jの窓のガラス部分の日射熱取得率η_(w,g,j) (φ)は次式で表される。
η_(w,g,j) (φ)=τ_(w,g,j) (φ)+(1-τ_(w,g,j) (φ)-ρ_(w,g,j) (φ))⋅r_(R,w,g,j)
η_(w,g,j) (φ)	：入射角φに対する境界jの窓のガラス部分の日射熱取得率, -
τ_(w,g,j) (φ)	：入射角φに対する境界jの窓のガラス部分の日射透過率, -
ρ_(w,g,j) (φ)	：入射角φに対する境界jの窓のガラス部分の日射反射率, -
r_(R,w,g,j)	：境界jの窓のガラス部分の日射吸収量に対する室内側に放出される量の割合, -

入射角φに対する境界jの窓のガラス部分の日射透過率τ_(w,g,j) (φ)及び入射角φに対する境界jの窓のガラス部分の日射反射率ρ_(w,g,j) (φ)は次式により表される。
τ_(w,g,j) (φ)={■(τ_(w,g,s1,j) (φ)&(単層の場合)@(τ_(w,g,s1,j) (φ)⋅τ_(w,g,s2,j) (φ))/(1-ρ_(w,g,s1b,j) (φ)⋅ρ_(w,g,s2f,j) (φ) )&(複層の場合) )┤
ρ_(w,g,j) (φ)={■(ρ_(w,g,s1f,j) (φ)&(単層の場合)@ρ_(w,g,s1f,j) (φ)+(τ_(w,g,s1,j) (φ)⋅τ_(w,g,s2f,j) (φ)⋅ρ_(w,g,s2f,j) (φ))/(1-ρ_(w,g,s1b,j) (φ)⋅ρ_(w,g,s2f,j) (φ) )&(複層の場合) )┤
ρ_(w,g,s1b,j) (φ)⋅ρ_(w,g,s2f,j) (φ)>0.9999の場合は、ρ_(w,g,s1b,j) (φ)⋅ρ_(w,g,s2f,j) (φ)=0.9999とする。（ゼロ割りの回避）

ρ_(w,g,j) (φ)	：入射角φに対する境界jの窓のガラス部分の日射反射率

ρ_(w,g,s1b,j) (φ)

ρ_(w,g,s2f,j) (φ)

単層の場合はτ_(w,g,s1,j) (φ)・ρ_(w,g,s1f,j) (φ)のみ計算すればよく、その他のτ_(w,g,s2,j) (φ)・ρ_(w,g,s1b,j) (φ)・ρ_(w,g,s2f,j) (φ)は複層のみ必要な値である。

τ_(w,g,s1,j) (φ)	
	：入射角φに対する境界jの窓のガラス部分の室外側から1枚目の板ガラスの透過率, -
τ_(w,g,s2,j) (φ)	
	：入射角φに対する境界jの窓のガラス部分の室外側から2枚目の板ガラスの透過率, -

境界 |j| の窓のガラス部分の日射熱取得率(η値) :math:`\eta_{w,g,j}` は次式により表される。

.. math::
    :nowrap:
    
    \begin{align*}
        \eta_{w,g,j} = \frac{ \eta_{w,j} }{ r_{A,w,g,j} }
    \end{align*}

| ここで
|   :math:`\eta_{w,g,j}` : 境界 |j| の窓のガラス部分の日射熱取得率, -
|   :math:`\eta_{w,j}` : 境界 |j| の窓の日射熱取得率(η値), -
|   :math:`r_{A,w,g,j}` : 境界 |j| の窓の面積に対するグレージングの面積の比, -
| である。

境界 |j| の窓のガラス部分の熱損失係数(U値) :math:`U_{w,g,j}` は次式により表される。

.. math::
    :nowrap:

    \begin{align*}
        U_{w,g,j} = \frac{ U_{w,j} - U_{w,f,j} \cdot ( 1 - r_{A,w,g,j} ) }{ r_{A,w,g,j} }
    \end{align*}

| ここで
|   :math:`U_{w,g,j}` : 境界 |j| の窓のガラス部分の熱損失係数(U値), W / |m2| K
|   :math:`U_{w,j}` : 境界 |j| の窓の熱損失係数(U値), W/m2K
|   :math:`U_{w,f,j}` : 境界 |j| の窓の建具部分の熱損失係数(U値), W / |m2| K
|   :math:`r_{A,w,g,j}` : 境界 |j| の窓の面積に対するグレージングの面積の比, -
| である。
